# 🔧 Documentación Técnica

Esta carpeta contiene documentación técnica del proyecto MLSM Smart Evacuation, separada de la memoria del TFG.

## Estructura

- **`schemas/`** — Especificaciones de datos
  - `mlsm_json_v1_schema.md` — Auditoría y especificación del schema JSON MLSM v1

- **`baseline/`** — Referencias de línea base
  - Documentación sobre versiones de referencia y cambios de versión

- **`architecture/`** — Decisiones de arquitectura
  - Documentación técnica sobre componentes del sistema

- **`decisions/`** — Architecture Decision Records (ADR)
  - Histórico de decisiones técnicas importantes

---

**Nota:** Esta carpeta NO contiene código de producción (ver `src/` en la raíz del proyecto).
