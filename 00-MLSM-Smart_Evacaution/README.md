## Archivos presentes
`MLSM_EvacEngine.py`: Es un simulador de evacuación para multitudes basada en Agentes (Agent Based Model) que utiliza la información de un json `IndoorData`, este simulador permite definir los periles de movilidad y da libertad a la hora de recomendar las rutas, mediante un sistema de dinamización de pesos en las aristas de la capa las baja `Door-to-Door`, tambien obtiene las metricas de la simuación para una auditoria posterior. 

`MLSM_SpatialEngine`: Esta es una herramienta avanzada de construcción basada en BIM que permite a aprtir de un lienzo construir la distribución de una planta interior `FloorPLant`, Permite construir a base ce clicks de forma totalmente libre asi como dejar un set de posiciones para los agentes a evacuas (que se unasra en el motor de evacuación). Esta propia herramienta Procesa y calcia las relaciones entre las distintas zonas, tambien permite asignar zonas por función y tipo de movilidad gracias a caracteristicas basadas en el estandar internaciona `OGC indoorGML`, Este procesador además de la geometría de una planta, calcula y obtiene el modelo Multi-Capas (`Multi-Layer Spatial Model`) donde las capas Con capas como **Adyacency Layer**, **Conectivity Layer**, **Door-to-Door Layer**
